
##

Also formulated as all possible sub-list/array partitions of a give array (non zero sized sublist/array)

https://www.geeksforgeeks.org/print-ways-break-string-bracket-form/

## Example

Input : abcd
Output : (a)(b)(c)(d)
         (a)(b)(cd)
         (a)(bc)(d)
         (a)(bcd)
         (ab)(c)(d)
         (ab)(cd)
         (abc)(d)
         (abcd)
